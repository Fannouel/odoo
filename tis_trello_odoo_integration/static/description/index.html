Trello - Odoo Integration

This module allows to integrate Odoo with Trello. This will manage projects and tasks from Trello to Odoo & Odoo to Trello.

Configuration

Go to preferences menu of the user and configure the trello API Key, API Token and Username.

Then click the 'Test URL' button to test the connection.

If the connection is success, you can see a success message.

After testing the connection click the 'Sync With Trello' button to sync datas bi-directionally.

Once the syncing is completed, you can see a success message.

After syncing you can see the projects from Trello in Odoo Projects.

Odoo Projects in Trello.

Trello Project View.

Odoo Project View.

Here you can see the description, due date, tags and attachments from trello card. By clicking 'Sync with Trello' button you can sync the task and project to Trello. Also you can track the last sync time with trello.
